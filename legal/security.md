# Security

This Security page is a placeholder for the Finzly API Developer Portal.

## Integration security basics

- Use TLS for all API calls.
- Store credentials and tokens in a secure secrets manager.
- Rotate keys and tokens regularly.
- Implement least privilege for service accounts.
- Monitor API usage and error spikes.
